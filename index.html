scuby by mcamou

An integration layer between Scala and JRuby

Currently the integration between Scala and JRuby has to go through the Java layer (JSR-223). The objective of this project is to make it as seamless as possible to call JRuby methods and functions from Scala and vice versa, to extend JRuby objects in Scala and so on.

Dependencies

Scala 2.7.7 JRuby 1.4.0

Install

mvn install
